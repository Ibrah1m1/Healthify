## 📌 Overview

**Healthify** is a **Java-based GUI application** that allows users to easily manage and monitor their health data. The application helps users track their **weight, height, and Body Mass Index (BMI)** over time, making it easier to maintain good fitness habits and notice health problems early.

---

## 🎯 Problem Statement

In today's busy lifestyle, many people find it difficult to track their health data such as weight, height, and body mass index (BMI). Without a simple way to record and monitor these details, it becomes hard to notice problems or maintain good fitness habits.

**The Problem:** Lack of a simple, user-friendly system that allows individuals to record, view, and monitor their health progress over time.

---

## 💡 Solution

Healthify provides:
- ✅ Simple and user-friendly interface
- ✅ Automatic BMI calculation
- ✅ Data storage in files for retrieval
- ✅ Visual tracking of health progress over time
- ✅ Easy access to previous records

## 📊 Features

| Feature | Description |
|---------|-------------|
| **BMI Calculator** | Automatically calculates BMI based on weight and height input |
| **Data Storage** | Stores user health data in files for future retrieval |

## 🚀 How to Run

```
bash
# Clone the repository
git clone https://github.com/Ibrah1m1/Healthify.git

# Navigate to project directory
cd Healthify

# Compile Java files
javac -d out src/*.java

# Run the application
java -cp out Main
```
